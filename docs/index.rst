TrainerDex Documentation
========================

.. note::
   TrainerDex is an evolving project and as such, this is a living document.

   You can help improve the documentation on `GitHub <https://github.com/TrainerDex/Website/tree/develop/docs>`_.

**TrainerDex** is a tool for tracking and comparing your Pokemon Go progress with other players.

As this documentation grows, we plan to cover everything TrainerDex can do, including;
  
  * **Tracking your progress**
  * **Generating leaderboards**
  * **Discord server management**
